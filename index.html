課程介紹

本課程是一套完整的 Python 學習教材，專為程式設計初學者與希望紮實掌握 Python 的學習者設計。 課程內容涵蓋從基礎語法到進階應用，包含資料結構、物件導向程式設計、檔案處理、資料分析等核心主題， 並透過大量實作練習與專案開發，幫助學習者建立紮實的程式設計能力。

課程特色

  • 16週系統化課程規劃
  • 循序漸進的學習路徑
  • 大量程式碼範例與實作
  • 完整的練習題與專案

學習時間

  • 建議每週7-10小時
  • 每天1-2小時持續學習
  • 完整課程約16週
  • 可依個人進度調整

適合對象

  • 程式設計初學者
  • 資訊相關科系學生
  • 需要學習 Python 者
  • 想紮實打好基礎者

學習成果

  • 掌握 Python 核心語法
  • 理解物件導向程式設計
  • 實作資料結構與演算法
  • 完成實際開發專案

課程大綱

第一階段：基礎入門 (Week 1-4)

建立 Python 程式設計的基礎知識，包含開發環境建置、基本語法、控制結構、資料型態與函數等核心概念。

Week 1
開發環境與基礎語法
Python 安裝與設定、第一個程式、變數與資料型態、基本運算、輸入輸出函數
Week 2
流程控制
條件判斷 (if-elif-else)、迴圈結構 (for, while)、break 與 continue、巢狀結構
Week 3
資料結構基礎
串列 (List)、元組 (Tuple)、字典 (Dictionary)、集合 (Set)、資料結構應用
Week 4
函數設計
函數定義與呼叫、參數傳遞、返回值、變數作用域、Lambda 函數、遞迴設計

第二階段：核心進階 (Week 5-8)

深入學習 Python 的核心特性，包含字串處理、檔案操作、異常處理、模組使用等進階主題。

Week 5
字串處理進階
字串方法、格式化字串、正規表示式、文字處理、編碼與解碼
Week 6
檔案處理
檔案讀寫、文字檔與二進位檔、JSON 與 CSV 處理、路徑操作、異常處理
Week 7
模組與套件
模組匯入、標準函式庫、第三方套件、pip 使用、自訂模組開發
Week 8
異常處理與除錯
try-except 結構、自訂異常、除錯技巧、日誌記錄、測試基礎

第三階段：物件導向 (Week 9-12)

探索物件導向程式設計，包含類別設計、繼承、多型、進階資料結構實作等主題。

Week 9
物件導向基礎
類別與物件、屬性與方法、建構函數、封裝概念、特殊方法
Week 10
繼承與多型
繼承概念、方法覆寫、多重繼承、抽象類別、介面設計
Week 11
進階資料結構
堆疊 (Stack)、佇列 (Queue)、鏈結串列、樹狀結構、圖形結構
Week 12
演算法設計
排序演算法、搜尋演算法、遞迴應用、動態規劃、時間複雜度分析

第四階段：實踐應用 (Week 13-16)

將所學知識應用於實際開發，包含資料分析、網路爬蟲、API 開發、以及完整的專案實作。

Week 13
資料分析基礎
NumPy 陣列運算、Pandas 資料處理、資料清理、統計分析、視覺化基礎
Week 14
網路程式設計
HTTP 請求、網路爬蟲、BeautifulSoup、API 串接、非同步程式設計
Week 15
綜合專案開發 (一)
專案需求分析、系統架構設計、資料庫整合、使用者介面設計
Week 16
綜合專案開發 (二)
學生成績管理系統：物件導向設計、檔案處理、資料分析、報表生成

基本執行流程

# 檢查 Python 版本
python --version
python3 --version

# 執行 Python 程式
python program.py

# 進入互動模式
python

# 安裝套件
pip install package_name

# 查看已安裝套件
pip list

# 更新套件
pip install --upgrade package_name

Windows 安裝步驟

  1. 從 python.org 下載最新版 Python 安裝程式
  2. 執行安裝程式，勾選 "Add Python to PATH"
  3. 選擇 "Install Now" 或自訂安裝
  4. 安裝完成後，開啟命令提示字元
  5. 輸入 python --version 確認安裝成功

macOS / Linux 安裝

# macOS (使用 Homebrew)
brew install python3

# Ubuntu / Debian
sudo apt update
sudo apt install python3 python3-pip

# Fedora
sudo dnf install python3 python3-pip

# 驗證安裝
python3 --version
pip3 --version

學習建議

學習方法

  • 理解概念後實際撰寫程式碼
  • 每週完成所有練習題
  • 嘗試修改範例程式碼
  • 善用 Python 互動模式測試

重要觀念

  • Python 強調程式碼可讀性
  • 縮排是語法的一部分
  • 善用內建函數和標準庫
  • 遵循 PEP 8 編碼風格

實作建議

  • 從小程式開始練習
  • 逐步增加程式複雜度
  • 建立個人專案作品集
  • 參與開源專案貢獻
