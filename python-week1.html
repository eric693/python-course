1. Python 簡介

Python 的歷史

Python 由 Guido van Rossum 於 1991 年創建，是一種高階、直譯式、物件導向的程式語言。Python 強調程式碼的可讀性，使用簡潔的語法，讓程式設計師可以用更少的程式碼表達想法。

為什麼學習 Python？

  • 易於學習：語法簡潔明瞭，接近自然語言，適合初學者
  • 應用廣泛：Web 開發、資料科學、機器學習、自動化、遊戲開發等
  • 豐富的生態系：有超過 30 萬個第三方套件可以使用
  • 跨平台：可在 Windows、macOS、Linux 等系統上執行
  • 社群活躍：龐大的開發者社群，容易找到學習資源
  • 職涯發展：是目前最受歡迎的程式語言之一

Python 的應用領域

領域 應用 常用套件
Web 開發 網站、API 開發 Django, Flask, FastAPI
資料科學 資料分析、視覺化 Pandas, NumPy, Matplotlib
機器學習 AI、深度學習 TensorFlow, PyTorch, scikit-learn
自動化 腳本、爬蟲 Selenium, Beautiful Soup, Requests
遊戲開發 2D 遊戲 Pygame
科學運算 數值計算、模擬 SciPy, SymPy

Python 的特點

優點：

  • 語法簡潔優雅，易於閱讀和維護
  • 開發速度快，提高生產力
  • 豐富的標準函式庫和第三方套件
  • 支援多種程式設計範式（物件導向、函數式等）
  • 自動記憶體管理（垃圾回收機制）
  • 強大的社群支援

需要注意：

  • 執行速度相對較慢（相比 C/C++）
  • 縮排是語法的一部分，需要特別注意
  • 動態型態可能導致執行時錯誤
  • 不適合開發手機應用程式

2. 開發環境安裝

安裝 Python

Windows 系統

  1. 前往 Python 官網：https://www.python.org/
  2. 點擊「Downloads」下載最新版本（建議 Python 3.8 以上）
  3. 執行安裝程式

  6. 安裝完成後，開啟命令提示字元（cmd）

基本資料型態

數值型態

# 整數 (int) age = 25 population = 1000000 negative = -50 # 浮點數 (float) pi = 3.14159 price = 99.99 scientific = 1.5e-4 # 科學記號：0.00015 # 複數 (complex) complex_num = 3 + 4j # 查看型態 print(type(age)) # print(type(pi)) # print(type(complex_num)) #

字串 (str)

# 單引號或雙引號都可以 name1 = 'Alice' name2 = "Bob" # 多行字串使用三引號 message = """這是一個 多行字串 可以跨越多行""" # 字串運算 greeting = "Hello" + " " + "World" # 連接 repeat = "Ha" * 3 # 重複：HaHaHa # 字串索引 text = "Python" print(text[0]) # P（第一個字元） print(text[-1]) # n（最後一個字元） print(text[0:3]) # Pyt（切片）

布林值 (bool)

# 只有 True 和 False 兩個值（注意大小寫） is_active = True is_completed = False # 比較運算會產生布林值 result1 = 10 > 5 # True result2 = 3 == 3 # True result3 = "a" != "b" # True

None（空值）

型態轉換

# 轉換成整數 num1 = int("123") # 字串轉整數：123 num2 = int(45.8) # 浮點數轉整數：45（無條件捨去） # 轉換成浮點數 num3 = float("3.14") # 字串轉浮點數：3.14 num4 = float(10) # 整數轉浮點數：10.0 # 轉換成字串 text1 = str(123) # 整數轉字串："123" text2 = str(3.14) # 浮點數轉字串："3.14" # 轉換成布林值 bool1 = bool(1) # True bool2 = bool(0) # False bool3 = bool("") # False（空字串） bool4 = bool("text") # True（非空字串）
型態轉換注意事項：
  • 不是所有字串都能轉換成數字，如 int("abc") 會產生錯誤
  • 浮點數轉整數會直接捨去小數部分，不會四捨五入
  • 數字 0、空字串、None 轉換成布林值都是 False

5. 基本運算符

算術運算符

# 基本運算 a = 10 b = 3 print(a + b) # 加法：13 print(a - b) # 減法：7 print(a * b) # 乘法：30 print(a / b) # 除法：3.3333...（結果是浮點數） print(a // b) # 整除：3（無條件捨去） print(a % b) # 餘數：1 print(a ** b) # 次方：1000 (10的3次方) # 複合運算 x = 10 x += 5 # x = x + 5，結果：15 x -= 3 # x = x - 3，結果：12 x *= 2 # x = x * 2，結果：24 x /= 4 # x = x / 4，結果：6.0 x //= 2 # x = x // 2，結果：3.0 x %= 2 # x = x % 2，結果：1.0 x **= 3 # x = x ** 3，結果：1.0

比較運算符

a = 10 b = 5 print(a == b) # 等於：False print(a != b) # 不等於：True print(a > b) # 大於：True print(a < b) # 小於：False print(a >= b) # 大於等於：True print(a <= b) # 小於等於：False # 字串比較 print("apple" < "banana") # True（按字母順序） print("10" > "2") # False（字串比較，非數字比較）

邏輯運算符

# and（且）：兩個都要為 True print(True and True) # True print(True and False) # False # or（或）：至少一個為 True print(True or False) # True print(False or False) # False # not（非）：反轉布林值 print(not True) # False print(not False) # True # 實際應用 age = 20 has_license = True # 檢查是否可以開車：年齡 >= 18 且 有駕照 can_drive = age >= 18 and has_license print(can_drive) # True

運算優先順序

優先級 運算符 說明
1（最高） ** 指數
2 *, /, //, % 乘除餘
3 +, - 加減
4 <, <=, >, >=, ==, != 比較運算符
5 not 邏輯 NOT
6 and 邏輯 AND
7（最低） or 邏輯 OR
# 運算優先級範例 result = 2 + 3 * 4 # 14（先乘後加） result = (2 + 3) * 4 # 20（括號優先） result = 2 ** 3 ** 2 # 512（右結合：2 ** (3 ** 2) = 2 ** 9） result = 10 > 5 and 3 < 7 # True # 建議：不確定時使用括號 result = (10 > 5) and (3 < 7) # 更清楚

6. 輸入與輸出

輸出：print() 函數

# 基本輸出 print("Hello, World!") # 輸出多個值 print("我的名字是", "Alice", "年齡是", 25) # 指定分隔符 print("Apple", "Banana", "Cherry", sep=", ") # 輸出：Apple, Banana, Cherry # 指定結尾符號（預設是換行） print("Hello", end=" ") print("World") # 輸出：Hello World # 格式化輸出 name = "Bob" age = 30 # 方法一：使用 f-string（推薦，Python 3.6+） print(f"我的名字是 {name}，年齡是 {age}") # 方法二：使用 format() print("我的名字是 {}，年齡是 {}".format(name, age)) # 方法三：使用 % 格式化（舊式） print("我的名字是 %s，年齡是 %d" % (name, age)) # 格式化數字 pi = 3.14159 print(f"圓周率約為 {pi:.2f}") # 保留兩位小數：3.14 print(f"百分比：{0.856:.1%}") # 百分比：85.6% # 對齊輸出 print(f"{'Name':<10} {'Age':>5}") # 左對齊、右對齊 print(f"{'Alice':<10} {25:>5}") print(f"{'Bob':<10} {30:>5}")

輸入：input() 函數

# 基本輸入（返回字串） name = input("請輸入你的名字：") print("你好，" + name) # 輸入數字需要型態轉換 age_str = input("請輸入你的年齡：") age = int(age_str) # 轉換成整數 print("明年你就", age + 1, "歲了") # 簡化寫法 age = int(input("請輸入你的年齡：")) # 輸入浮點數 height = float(input("請輸入你的身高（公分）：")) print(f"你的身高是 {height} 公分") # 輸入多個值 data = input("請輸入三個數字（用空格分隔）：") a, b, c = data.split() # 分割字串 print(f"你輸入的數字是：{a}, {b}, {c}")
input() 注意事項：
  • input() 永遠返回字串型態
  • 如果需要數字，必須使用 int() 或 float() 轉換
  • 轉換時如果輸入格式錯誤會產生錯誤

完整範例

# 簡單的計算機程式 print("=== 簡單計算機 ===") # 輸入兩個數字 num1 = float(input("請輸入第一個數字：")) num2 = float(input("請輸入第二個數字：")) # 計算 sum_result = num1 + num2 diff_result = num1 - num2 prod_result = num1 * num2 div_result = num1 / num2 if num2 != 0 else "無法除以零" # 輸出結果 print(f"\n計算結果：") print(f"{num1} + {num2} = {sum_result}") print(f"{num1} - {num2} = {diff_result}") print(f"{num1} × {num2} = {prod_result}") print(f"{num1} ÷ {num2} = {div_result}")

7. 註解與編碼風格

註解

# 這是單行註解 # Python 使用 # 符號表示註解 # 可以在程式碼後面加註解 x = 10 # 設定 x 的值為 10 """ 這是多行註解 可以跨越多行 通常用於函數或模組的說明文件 """ ''' 也可以使用單引號 建立多行註解 ''' # 好的註解範例 def calculate_bmi(weight, height): """ 計算 BMI 值 參數: weight: 體重（公斤） height: 身高（公尺） 返回: BMI 值（浮點數） """ return weight / (height ** 2)

編碼風格（PEP 8）

命名規則

# 變數和函數：使用小寫字母和底線 user_name = "Alice" total_count = 100 def calculate_total(): pass # 常數：全部大寫 MAX_SIZE = 100 PI = 3.14159 # 類別：使用駝峰式命名（每個單字首字母大寫） class StudentManager: pass # 私有變數：開頭使用底線 _private_var = "secret" # 避免使用的名稱 # l（小寫 L）、O（大寫 O）、I（大寫 i）容易混淆

縮排

# Python 使用縮排表示程式碼區塊 # 建議使用 4 個空格（不要使用 Tab） # 正確的縮排 if True: print("這是縮排的程式碼") print("同一層級的程式碼") # 巢狀縮排 if True: print("第一層") if True: print("第二層") if True: print("第三層")

空白

# 運算符兩側加空格 x = 10 + 20 # 正確 x=10+20 # 不推薦 # 逗號後面加空格 items = [1, 2, 3, 4] # 正確 items = [1,2,3,4] # 不推薦 # 函數參數 def greet(name, age): # 正確 pass # 不要在括號內加空格 print("Hello") # 正確 print( "Hello" ) # 不推薦

每行長度

# 每行不超過 79 個字元 # 如果太長，可以使用反斜線或括號換行 # 方法一：使用反斜線 total = very_long_variable_name + \ another_long_variable_name + \ yet_another_variable # 方法二：使用括號 total = (very_long_variable_name + another_long_variable_name + yet_another_variable) # 方法三：字串換行 message = ("這是一個很長的訊息，" "我們把它分成多行" "讓程式碼更容易閱讀")
良好的編碼習慣：
  • 使用有意義的變數名稱
  • 保持一致的縮排（建議 4 個空格）
  • 適當添加註解說明複雜邏輯
  • 每行不要太長（建議不超過 79 字元）
  • 函數之間空兩行，類別定義之間空兩行
  • 使用空行分隔邏輯區塊

8. 常見錯誤與除錯

語法錯誤（SyntaxError）

# 錯誤：忘記冒號 if x > 5 print("x 大於 5") # 正確 if x > 5: print("x 大於 5") # 錯誤：縮排不一致 if True: print("第一行") print("第二行") # 縮排不一致 # 正確 if True: print("第一行") print("第二行") # 錯誤：引號不匹配 text = "Hello' # 正確 text = "Hello" # 或 text = 'Hello'

名稱錯誤（NameError）

# 錯誤：使用未定義的變數 print(message) # NameError: name 'message' is not defined # 正確：先定義再使用 message = "Hello" print(message) # 錯誤：變數名稱拼錯 user_name = "Alice" print(username) # NameError # 正確 print(user_name)

型態錯誤（TypeError）

# 錯誤：不同型態無法直接運算 age = 25 message = "我今年 " + age + " 歲" # TypeError # 正確：轉換成字串 message = "我今年 " + str(age) + " 歲" # 或使用 f-string message = f"我今年 {age} 歲" # 錯誤：對不可變型態進行修改 text = "Hello" text[0] = "h" # TypeError: 字串不可變 # 正確：建立新字串 text = "h" + text[1:]

值錯誤（ValueError）

# 錯誤：型態轉換失敗 number = int("abc") # ValueError: invalid literal for int() # 正確：確保可以轉換 text = "123" number = int(text) # 錯誤：分割數量不符 a, b = "1 2 3".split() # ValueError: too many values to unpack # 正確：數量要對應 a, b, c = "1 2 3".split()

除錯技巧

# 1. 使用 print() 除錯 x = 10 y = 20 print(f"x = {x}, y = {y}") # 檢查變數值 # 2. 使用 type() 檢查型態 data = "123" print(type(data)) # # 3. 分段測試 # 將程式分成小段落逐步測試 # 4. 閱讀錯誤訊息 # 錯誤訊息會指出錯誤位置和原因 # 5. 使用 try-except 處理例外（後面會學到） try: number = int(input("請輸入數字：")) except ValueError: print("輸入格式錯誤！")
除錯建議：
  • 仔細閱讀錯誤訊息，它會告訴你問題在哪
  • 使用 print() 檢查變數的值和型態
  • 將程式分成小段落逐步測試
  • 檢查縮排是否正確
  • 確認變數名稱沒有拼錯
  • 注意大小寫（Python 區分大小寫）

本週練習題

基礎練習（必做）

  1. 個人資訊輸出
    撰寫程式，輸入你的姓名、年齡、身高、體重，然後格式化輸出。
    # 預期輸出格式 姓名：王小明 年齡：20 歲 身高：175.5 公分 體重：65.0 公斤
  2. 溫度轉換
    輸入攝氏溫度，轉換成華氏溫度並輸出。
    公式：華氏 = 攝氏 × 9/5 + 32
  3. BMI 計算器
    輸入身高（公分）和體重（公斤），計算並輸出 BMI 值。
    公式：BMI = 體重(kg) / (身高(m))²

進階練習

  1. 圓面積與周長
    輸入圓的半徑，計算並輸出面積和周長。
    提示：可以使用 math.pi 或自訂 π = 3.14159
  2. 時間換算
    輸入總秒數，轉換成「小時:分鐘:秒數」格式。
    例如：3665 秒 → 1 小時 1 分鐘 5 秒
  3. 購物結帳
    輸入商品單價、數量、折扣率（如 0.9 表示九折），計算：
    • 小計金額
    • 折扣後金額
    • 輸入付款金額，計算找零

挑戰練習

  1. 年齡計算機
    輸入出生年份，計算今年幾歲，並顯示：
    • 實歲
    • 虛歲（實歲+1）
    • 距離下一個十年生日還有幾年
  2. 貨幣換算
    輸入台幣金額和匯率，計算可兌換的外幣金額。
    同時計算手續費（例如收取 1% 手續費）。
