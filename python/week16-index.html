Week 16 - Python 綜合專案開發

學生成績管理系統 - 整合所有 Python 核心技能

課程概述

本週課程將帶領你開發一個完整的學生成績管理系統，整合 Python 學習過程中的所有核心技能。透過這個專案，你將學會如何：

  • 設計物件導向的程式架構
  • 實作完整的資料管理功能（CRUD）
  • 處理檔案資料的儲存與讀取（JSON）
  • 建立使用者友善的命令列介面
  • 實作統計分析與報表功能
  • 設計完善的異常處理機制
專案特色：
  • 完整的專案開發流程
  • 模組化的程式設計
  • 實用的資料管理系統
  • 可擴充的系統架構
  • 專業的程式碼品質

課程內容

Part 1
Student 類別設計
學習物件導向程式設計，設計學生類別，實作基本屬性和方法，包含成績計算、等級判定、資料轉換等功能
Part 2
StudentManager 類別
實作學生管理系統核心功能，包括 CRUD 操作、JSON 資料持久化、統計分析、CSV 匯出、備份機制等
Part 3
主程式設計
設計命令列使用者介面，整合所有功能模組，實作選單系統、輸入驗證、錯誤處理，完成完整系統
Part 4
練習與總結
提供練習題、專案延伸建議、開發最佳實務、學習總結，幫助你深化學習成果並規劃未來方向

專案檔案

核心程式檔案

  • student.py - Student 類別定義（約 120 行）
  • student_manager.py - StudentManager 類別（約 300 行）
  • main.py - 主程式與使用者介面（約 400 行）

資料檔案

  • data/students.json - 學生資料儲存
  • data/backup/ - 資料備份目錄
  • reports/ - 報表輸出目錄
開始學習：

建議按照 Part 1 → Part 2 → Part 3 → Part 4 的順序學習，每個部分都包含完整的程式碼和詳細說明。
